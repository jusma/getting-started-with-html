The Basics of the Web:

The web is made up of HTML, urls, https, and web applications. The world wide web (www) is made up of HTML documents.

HTML stands for hypertext markup language and is the main protocol of the web.

Servers are computers that host the files that make up the web.

Internet is the world's largest computer network.

Browser is a program that runs on your computer to display files found on the web.

HTML is made up of: text content, markup, references to other documents, links to other pages.

Basic HTML Vocabulary:

Here are some basic HTML words that will be helpful to know along the way.

Tags are the commands within angled brackets that tell the computer what to do. There are two types of tags, an open and closing tag.

Elements refer to anything within and open and closing tag.

Attribute is a property of an HTML element. Used in the placement of a url link for a video, for example.
